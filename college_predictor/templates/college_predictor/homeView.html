Why use College Predictor ?

Confused about how you can make best use of the efforts you put in during class 12? Still pondering over college choices? Don't worry we are there for you

After attempting JEE Mains and Advanced on successful completion of class 12, aspirants are provided with the option of selecting a college from several colleges. Too much of anything is just as bad as too less of anything. The same holds good for choices of colleges as well.
Our college predictor attempts to sort out colleges according to user preferences, college reputation(NIRF rank) and also provide the user with basic details about the college. In this predictor, the user has to enter their branch preference order through select boxes. The user will also have to enter their rank, category rank( if they belong to any category), home- state, etc. The system will generate the result according to data input. After logging in user can provide feedback about a specific college.
The user interface is extremely friendly. A rank predictor is also provided to approximate rank. Data entry is through the admin panel.
